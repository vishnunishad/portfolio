# Portfolio

Personal portfolio website for Vishnu Nishad.
[MY WEBSITE](vishnunportfolio.netlify.app)

## Project Contents

- `index.html` - Root entrypoint for static hosts (redirects to portfolio page)
- `vishnu.html` - Main portfolio webpage (single-file HTML/CSS/JS)
- `vishnuResume.pdf` - Resume document
- `vercel.json` - Vercel routing configuration

## Features

- Modern animated UI with custom typography
- Light/Dark theme support
- Interactive sections for skills and profile highlights
- Responsive layout for desktop and mobile

## Run Locally

This is a static project. Open `index.html` directly in your browser.

## Deployment

You can deploy this project on any static hosting platform, such as:

- GitHub Pages
- Netlify
- Vercel (static)
